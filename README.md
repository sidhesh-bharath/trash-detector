# ♻️ YOLO Trash Detection Model

A trash detection model that can identify **paper**, **cardboard**, **metal**, **biodegradable**, and **glass** waste.  
It is designed for **EcoSweep**, an autonomous trash-collecting robot that automatically segregates waste at the source.

---

## Usage

1. Place your trained model file (or use the existing one) `best.pt` and your input image `image.png` — in the same directory.
2. Install dependencies:
   ```bash
